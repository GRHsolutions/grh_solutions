# GRH solutions
- Nuestro software se desarrolla en la empresa Disolin, la cual se dedica a la distribución y monitoreo de máquinas expendedoras. Disolin enfrenta problemas en la gestión de recursos humanos, que actualmente recae en una sola persona, lo que resulta en varios procesos manuales y poco eficientes. La cual se refleja en pérdida de documentos, lentitud en la contratación, y falta de un sistema centralizado para la planificación de actividades y horarios. Las causas de estos asuntos radican principalmente en la falta de organización y centralización de la información en la gestión de recursos humanos. Al no contar con un sistema que estructure y centralice adecuadamente los datos del personal, los procesos se vuelven manuales e ineficientes. Las consecuencias son pérdida de tiempo, demoras en el proceso de contratación  y la planificación de tareas y horarios también se ve afectada.

# IMPORTANTE PARA EJECUCIONES RAPIDAS 
1. Ejecutar el siguiente comando en la terminal de git bash 'bash execute.sh'(en la carpeta principal donde esta la carpeta test.client y test.server), esto ejecutara ambos proyectos al mismo tiempo, abriendo dos terminales de powershell nuevas, click si no tiene instalado git bash.
2. Ejecutar el siguiente comando en la terminal de git bash 'bash install.sh'(en la carpeta principal donde esta la carpeta test.client y test.server), esto ejecutara el comando npm i en ambas carpetas instalando las dependencias para que ambas funcionen correoctamente.
